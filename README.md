# CryoMap

Aplicativo web para monitoramento de temperatura de salas, gestão de 
equipamentos de refrigeração, tarefas técnicas, atendimentos, tempo parado, uploads e relatórios.

## Tecnologias previstas

### Frontend

- React
- TypeScript
- Vite
- Axios
- Recharts

### Backend

- NestJS
- TypeScript
- Node.js
- Prisma
- JWT
- bcrypt
- Multer

### Banco de dados

- PostgreSQL

### Ambiente

- Docker
- Docker Compose
